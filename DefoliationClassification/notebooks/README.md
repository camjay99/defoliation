Purpose of the notebooks in this directory:
data_download.ipynb - Preprocess (add VIs) and download data from Google Earth Engine
defol_map.ipynb - Fit models to each of the pixels in the region
defol_statistics.ipynb - Calculate defoliation metrics based off seasonal trend models
defol_statistics_2.ipynb - Classify image and create figures

Other files are for other analyses still taking place. This will be updated as things become more solidified.
